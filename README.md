# SoClo
[SoClo](http://soundcloud.com)

## Minimum Viable Product
- [ ] Create / update an account
- [ ] Log in / Log out
- [ ] Follow users
- [ ] Songs
  - [ ] Upload
  - [ ] Play songs
  - [ ] Tag songs
  - [ ] Comment on songs
  - [ ] Like songs
- [ ] Search for users
- [ ] View feed from followed users
- [ ] Searching by song title
- [ ] Filter by tags
